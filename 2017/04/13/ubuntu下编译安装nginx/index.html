ubuntu下编译安装nginx

Ubuntu下编译安装nginx

  • 参考这里

  • 安装完成之后，编辑修改nginx.conf， 路径是

    /usr/local/nginx/conf/nginx.conf

  • 启动nginx 

    sudo /usr/local/nginx/sbin/nginx

  • 关闭nginx

    sudo /usr/local/nginx/sbin/nginx -s stop

  • 重读nginx配置文件

    sudo /usr/local/nginx/sbin/nginx -s reload
